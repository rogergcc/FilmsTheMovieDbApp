# FilmsTheMovieDbApp

Android app fetch Movies from Api

- [TMDB API](https://developer.themoviedb.org/docs) - using TMDB API v3 Stable to fetch movies

need a key in local.properties

```api_key="YOUR_KEY"```

MVVM
